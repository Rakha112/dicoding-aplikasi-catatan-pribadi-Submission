## Run Locally

Clone Projek

```bash
  git clone https://github.com/Rakha112/dicoding-aplikasi-catatan-pribadi-submission.git
```

Perbi ke Projek Folder

```bash
  cd dicoding-aplikasi-catatan-pribadi-submission
```

Install dependencies

```bash
  npm install
```

Start Aplikasinya

```bash
  npm start
```
